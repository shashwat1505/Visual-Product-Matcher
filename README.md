# Visual-Product-Matcher: 
A web-based application that identifies and matches visually similar products using image processing and machine learning techniques. Users can upload a product image and receive the most visually similar products from a predefined image dataset.

---

 # Project Description

The **Visual Product Matcher** enables visual search by comparing product images based on their visual features. This project is useful for e-commerce platforms, recommendation systems, and duplicate product detection.

---

# Features

- Upload a product image for visual matching
- Matches against a dataset of 50+ product images
- Displays top visually similar products
- Uses image feature extraction techniques
- Simple and user-friendly interface
- Deployed with a working URL

---

# Tech Stack

- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** Python (Flask)  
- **Image Processing:** OpenCV  
- **Machine Learning:** CNN / Feature Extraction  
- **Dataset:** Local Image Dataset (50+ images)
